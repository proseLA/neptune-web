Neptune is the Design System built by and used at Wise. This is the Neptune framework for Web. The React components and CSS provide a way to build high quality, consistent user experiences on the web with ease.

## Setup

The React components and CSS are distributed separately as npm packages. Get started now by following the <DocumentLink href="/about/Setup">setup guide</DocumentLink>.

## Components

Neptune Web components contains many widgets ranging from simple buttons to more complex components like typeahead or flow navigation. Browse all our <DocumentLink href="/components/inputs/Button">components</DocumentLink>, or feel free to raise an issue or contribute if there's something else you'd like to see.

## CSS

Neptune Web CSS provides core foundational styles like typography, colours and spacing, with a number of utility classes for layout, display and more. Read the <DocumentLink href="/about/Styles">style documentation</DocumentLink> for in-depth information about styling your application with Neptune CSS.

## Dynamic forms

Neptune Web Dynamic forms provides form related components and dynamic form functionality. All dynamic form <DocumentLink href="/components/dynamicForms/DynamicFieldDefinitionList">components</DocumentLink> are listed under the grouping Dynamic forms.

## Usage guidelines

Platform agnostic usage guidelines are available on [Github](https://github.com/transferwise/neptune). If you have questions or comments on component usage you can raise them [there](https://github.com/transferwise/neptune/issues).

## Page Direction

Neptune Web (`@transferwise/neptune-css` and `@transferwise/components`) supports both LTR and RTL page directions.

Atm we enable RTL mode for following locales:

- Arabic (`ar`)
- Hebrew (`he`)

For more info dive into <DocumentLink href="styles/addons/RtlLayout">RTL layout</DocumentLink>.
