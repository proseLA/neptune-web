November Release

November 30, 2021
    in 12 days by Naiara Abaroa, Mate Bek, and Anton Dozortsev

    Updated Typography

    Neptune Design System made cross platform alignment and improved consistency for typography styles. Learn more about that here. We updated styles of components and Neptune CSS: font size, font weight, line height and emphasis styles. Headings are going to keep the same size for mobile and desktop.

  • To keep former h1 size, specially in marketing pages, you can use `.display-5` utility class.

    • We also adding new CSS utility classes that have cross platfrom naming.

    Support for RTL mode

    And last but not least we made all our components and styles (neptune-css) work in both page direction. Also, we provide guidance and tools for consumers to enable support for RTL mode on their side.

    Migration Checklist

    1. Update Neptune and all UI dependencies (make sure all UI dependencies have migrated to this version or higher version of components (39.x) and neptune-css (12.x)).
    2. Use new colors incase you have custom color styles (check .less, .css files) in your repository.
      If you wish to use CSS custom properties and you need provide support for IE11 you need to use PostCSS and @arshaw/postcss-custom-properties plugin (setup and more info here).

      Tip: Colors have semantic names so pick color variables based on CSS property.
      e.g if CSS property is background-color pick one of background colors tokens
      , if color choose of content (text and icon) tokensincase it's border-color use border color tokensor in some casesinteractive
      incase its :hover pseudo class use use tokens with -hover suffix, for :focus / :active use tokens with -active.

      Note: Mostly you would need to use default colors (light theme) variables, incase you have some part of UI with navy theme please use variables with -navy- part in name.

    3. Please run application/dev envirnment and perform visual review to see whether colors and typography are correct and things looks good.
