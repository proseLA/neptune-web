Markdown

Accepts markdown strings and returns html markup. The markdown parser used is commonmark

**- Headings**
# Heading1
## Heading2
### Heading3
#### Heading4   
***
  
**- Emphasis**  
*This text will be italic*  
_This will also be italic_  
**This text will be bold**  
__This will also be bold__
***
**- Lists**
* Item 1
* Item 2
  * Item 2a
  * Item 2b  
***
**- Inline code**
I think you should use an
`<addr>` element here instead
***
  
**- Blockquotes**
> Oh well this is an 
> amazing blockquote
***
